HTTP Basic Authentication

Overview

HTTP Basic Authentication is a simple authentication scheme built into the HTTP protocol. The client sends HTTP requests with the Authorization header that contains the word Basic followed by a space and a base64-encoded string username:password.

Implementation Details

  • Header Format: Authorization: Basic
  • Credentials: base64(username:password)
  • Security: Requires HTTPS to prevent credential interception
  • Session: Stateless - credentials sent with every request

Use Cases

  • API authentication
  • Simple web services
  • Internal systems with trusted networks
  • Quick prototyping and development

Testing

You can test this endpoint using curl:

curl -u user:password http://localhost:8080/basic/home
