Custom Configuration Instructions

To use a custom Anypoint Exchange, you need to create a Connected App in Anypoint Platform with the following specifications:

  • App Type: App acts on its own behalf (Client Credentials)
  • Required Permissions:
    • API Manager: View APIs Configuration
    • Exchange: Exchange Viewer
    • General: View Environment

Important: The exchange integration will only retrieve A2A Agents that are managed under Flex Gateway.
